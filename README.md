# SQL AIFA Project

This repository contains a Python-based academic project related to AIFA pharmaceutical data processing and structured data handling.

## Project Overview
The project focuses on collecting, organizing, and processing pharmaceutical information in a structured format using Python and SQL-oriented data workflows.

## Main Tasks
- Read and process pharmaceutical data
- Organize extracted information into a structured format
- Prepare data for storage and analysis
- Support SQL/database-oriented workflows

## Tools and Libraries
- Python
- Pandas
- SQL-oriented data processing
- File handling and automation utilities

## Repository Structure
- `src/main.py` → main project script
- `data/` → input or output data files
- `docs/` → project notes, report, or screenshots
- `requirements.txt` → required Python libraries
- `README.md` → project documentation

## How to Run
Install the required libraries first:

```bash
pip install -r requirements.txt
python src/main.py
